# Vending-Machine-Management

 


A simple program that allows you to make your own Vending Machine Management service, all in terminal, where you can create as many vending machines as you want.

# Execution

1) Make sure all code is in the same folder
2) Run the "break_room_complex.py" file
3) Use the program to your hearts content! Everything should have exception handlings to make sure no errors arise:

# Features:
1) Make Vending Machine : Allows you to make a vending machine with as many items you want with the following specifications [name, price, quantity]
2) Use Vending Machine : Allows you to use your vending machines with a second menu:
3) Add Money : Allows you to add any denomination from a nickel to $20, nothing else will be officially processed
4) Vend : Asks for item you want to be vended, must have the money and the quantity
5) Refund : Refunds the credits currently in the vending machine
6) Return : Returns to first menu
7) Quit : Quits the program
